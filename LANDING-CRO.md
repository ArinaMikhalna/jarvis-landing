## Секционный анализ

### 1. Hero-секция — 8/10 (вес 25%)

**Находки:**
- CTA и трасты видны без скролла и на 1280×800, и на мобиле ✓
- Заголовок проходит тест «So What»: боль (теряешь деньги) + решение (Claude) + цена альтернативы (час и 20 $)
- Персонаж + реактор = мгновенный мэтч с метафорой курса; переключатель «Броня/Claude-ДЖАРВИС» — приятная деталь
- Трасты под кнопкой: доступ сразу, возврат 7 дней, разовый платёж ✓
- ❌ Выше fold НЕТ соцдоказательства (ни числа учеников, ни отзыва)
- ⚠️ Boot-экран (2–3 сек) — трение до первого касания для холодного трафика (показывается 1 раз за сессию — смягчено)
- ⚠️ Ключевое обещание «за один вечер» спрятано в подзаголовке

**Исправления:**

| Приоритет | Изменение | Ожидаемый лифт |
|-----------|----------|---------------|
| ВЫСОКИЙ | Добавить строку соцпруфа под трастами: «N учеников уже собрали своего ДЖАРВИСА» (реальное число) | +5–10% |
| СРЕДНИЙ | A/B: вынести «за один вечер» в H1 — «Собери за один вечер ДЖАРВИСА, который закрывает твою рутину» | +10–20% CTR hero |
| НИЗКИЙ | A/B: сократить boot до ~1.2 сек для платного трафика | −bounce 2–5% |

### 2. Ценностное предложение — 8/10 (вес 20%)

**4U:** Useful 9 · Urgent 4 · Unique 8 · Ultra-specific 8

**Находки:**
- «10% → 100%» — сильная визуальная драматизация недоиспользования
- Облако из 24 задач — сканируемо, закрывает вопрос «а что конкретно он умеет»
- Дифференциация есть и она уникальна: мета-блок «ты внутри пруфа» (сайт собран этим же методом)
- ❌ Срочность после удаления таймера — 4/10: нет причины купить сегодня

**Исправления:**

| Приоритет | Изменение | Ожидаемый лифт |
|-----------|----------|---------------|
| ВЫСОКИЙ | Вернуть ЧЕСТНУЮ срочность: «Чат с разбором задач стартует [дата] — успей к старту» или бонус до даты | +5–15% |

### 3. Социальное доказательство — 6/10 (вес 15%) ← главный резерв

**Находки:**
- 7 реальных ТГ-скринов отзывов + листалка ✓
- Кейсы автора с цифрами (10 мин, −70% операционки) ✓
- ❌ Первый пруф появляется только на ~10-м экране
- ❌ Нигде нет ЧИСЛА учеников
- ❌ Блок автора без цифр авторитета (опыт, клиенты агентства, аудитория)
- ⚠️ Отзывы без подписей «кто это» (роль) — теряется мэтч с сегментами ЦА

**Исправления:**

| Приоритет | Изменение | Ожидаемый лифт |
|-----------|----------|---------------|
| ВЫСОКИЙ | Соцпруф выше fold (строка с числом учеников) | +5–10% |
| ВЫСОКИЙ | Мини-полоса пруфа после блока «Проблема»: 1 сильный отзыв-цитата + число | +3–8% |
| СРЕДНИЙ | Автору добавить 2–3 факта с цифрами (лет в маркетинге, клиентов, подписчиков) | +3–5% |
| НИЗКИЙ | Подписи к отзывам: «маркетолог», «эксперт-психолог» — мэтч с ЦА | +2–4% |

## Аудит формы

Форм нет — оплата через Telegram-бот (переход по кнопке). Микрокопия о переходе
есть ✓. Риск разрыва цепочки (сайт → ТГ) частично снят, но:

- ❌ **Клики по кнопкам оплаты НЕ отслеживаются** (нет `ym reachGoal`) —
  конверсию сайт→бот сейчас измерить нечем. Это исправление №1 по важности
  для управления: без него все лифты — вслепую.

## Мобильный аудит

- [x] CTA thumb-reachable, sticky-кнопка
- [x] Текст ≥16px, переносы вычищены (без слоговых, без висячих предлогов)
- [x] Нет горизонтального скролла
- [x] Тап-таргеты ≥44px
- [x] Хедер/загрузчик адаптированы под Telegram-webview
- [ ] Скорость на 4G — см. ниже

## Оценка скорости

| Проблема | Факт | Решение |
|----------|------|---------|
| hero.png 1.4 MB | LCP на 4G ~3–5 сек → −7…−20% CR | Конверт в WebP (~250–400 KB) |
| avatar 683 KB, ru-hero 609 KB, who-* по ~530 KB | всё грузится сразу | `loading="lazy"` на ВСЕ img ниже fold |
| Суммарно картинок ~5+ MB | тяжёлая первая загрузка | WebP + lazy = стр. быстрее в 2–3 раза |

Скрипты уже в конце body (не блокируют) ✓. Кэширование статики настроено ✓.

## A/B тест-гипотезы (топ-5)

1. **Если** добавить соцпруф выше fold (число учеников + звёзды), **то** CR вырастет,
   **потому что** первый пруф сейчас на 10-м экране. Лифт 5–15% · Низкая сложность
2. **Если** вынести «за один вечер» в H1, **то** вырастет глубина скролла и CR,
   **потому что** срок результата — сильнейший элемент оффера. Лифт 10–20% · Минимальная
3. **Если** связать калькулятор с ценой («рутина стоит тебе X ₽/мес — ДЖАРВИС 4990 разово»),
   **то** CR тарифов вырастет, **потому что** цена получает динамический якорь. Лифт 5–10% · Средняя
4. **Если** добавить 40-сек видео-демо после «Проблемы», **то** CR вырастет,
   **потому что** «покажи, не рассказывай» снимает недоверие к обещаниям. Лифт 10–25% · Средняя
5. **Если** вернуть честный дедлайн (старт чата разбора / бонус до даты),
   **то** CR вырастет, **потому что** сейчас срочность 4/10. Лифт 5–15% · Низкая

## Тепловая карта (прогноз)

- Максимум внимания: H1 → лицо/реактор персонажа → кнопка. Персонаж смотрит в кадр —
  ок, реактор ведёт взгляд к CTA-колонке ✓
- Риск dead zone: длинный пиннинг «сборки костюма» на десктопе (470vh) — часть
  пользователей выйдет из пиннинга усталой; CTA-полосы до/после смягчают ✓
- Rage-click риск: плашки-«капсулы» в облаке задач выглядят кликабельными, но не кликаются
  (низкий вред)

## Приоритизированный план

### Быстрые победы (эта неделя)
1. **Цели Метрики** на клики кнопок оплаты (`reachGoal`) — без этого нельзя мерить вообще
2. **WebP + lazy loading** всех картинок — самый надёжный «технический» лифт (+7–20% на мобиле)
3. **Число учеников** в hero + мини-пруф после «Проблемы» (+5–15%)
4. Подписи-роли к отзывам (+2–4%)

### Среднесрочные (этот месяц)
1. Честная срочность: дата старта чата разбора / бонус до даты (+5–15%)
2. Динамический якорь цены из калькулятора (+5–10%)
3. Цифры авторитета в блок автора (+3–5%)
4. A/B заголовка hero («за один вечер» в H1)

### Стратегические (квартал)
1. Видео: 40-сек демо «Claude делает работу» или видеообращение Арины (+10–25%)
2. Скрины/GIF внутрянки курса в блок программы
3. Прелендинг под холодный платный трафик
